dar preferencia pra semantica quando utilizar HTML

Negrito/ Destaque

Nesta frase temos um termo em negrito usando a tag B (nao semantica)

Nesta frase temos, termo em destaque usando a tag STRONG (semantica)

Italico/ Enfase

Nesta frase, temos um termo em italico usando a tag I (nao semantico)

Nesta frase temos um termo em enfase usando a tag EM (semantico)

Texto Marcado

Podemos criar tambem um texto marcado usando a tag MARK(marca texto)

Texto Grande/ Pequeno

Estamos criando um Texto Grande e um Texto Pequeno com a TAG BIG/SMALL

Texto Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas nao considerado com a TAG DEL

Texto Inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depoois com a TAG INS (semantico)

Texto Sublinhado

Podemos marcar um texto como sublinhado para dar uma enfase com a TAG U (nao semantica)

Texto Subscrito/ Sobrescrito

Existe tambem como deixar subscrito para coisas como H20 que fica H20 e mibr com as TAGS SUP (para cima) e SUB (para baixo)
